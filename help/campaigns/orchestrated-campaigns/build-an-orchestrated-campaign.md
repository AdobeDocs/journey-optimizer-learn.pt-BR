# Criar uma campanha orquestrada

Saiba como projetar e executar campanhas orquestradas no Adobe Journey Optimizer, criando fluxos de trabalho que combinam públicos, manipulação de dados e entrega de vários canais. Este vídeo aborda a programação de campanha, a criação de público-alvo a partir de dados relacionais, o enriquecimento e a desduplicação e o uso de vários fluxos para oferecer suporte a casos de uso avançados, como acompanhamentos de carrinho abandonado, prevenção de churn e notificações de retorno ao estoque.
