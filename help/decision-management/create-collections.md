# Criar coleções

Saiba como criar coleções para o gerenciamento de decisões.

As coleções são usadas para gerenciar ofertas em grupos lógicos e são necessárias para criar atividades de gerenciamento de decisões. As coleções dinâmicas permitem agrupar ofertas com base em tags, enquanto as coleções estáticas permitem preparar manualmente as ofertas.
