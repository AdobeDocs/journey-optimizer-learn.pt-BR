# Criar ofertas substitutas

Saiba como criar [ofertas substitutas] (https://experienceleague.adobe.com/docs/journey-optimizer/using/offer-decisioniong/managing-offers-in-the-offer-library/creating-fallback-offers.html) no [!DNL Offer Decisioning]. As ofertas substitutas são ofertas padrão exibidas para clientes que não se qualificam para nenhuma das ofertas personalizadas.
