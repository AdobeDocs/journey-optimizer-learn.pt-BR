# Monitorar ações personalizadas

Saiba como monitorar com eficiência a integridade operacional e o desempenho de chamadas HTTP feitas de suas jornadas para pontos de acesso de terceiros por meio de ações personalizadas. Obtenha visibilidade de erros de HTTP, gargalos de latência e problemas de limite que podem afetar a entrega.

Este relatório ajuda a identificar problemas, melhorar o desempenho da jornada e garantir uma operação tranquila.
