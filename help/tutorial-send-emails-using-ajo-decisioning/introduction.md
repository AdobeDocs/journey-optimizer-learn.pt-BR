# Usar o Decisioning para personalizar emails

Neste tutorial, exploraremos um caso de uso real de envio de ofertas de email personalizadas usando o Adobe Journey Optimizer Decisioning. O cenário é simples: enviar ofertas de email direcionadas aos clientes com base nos atributos do perfil, como renda anual e status de fidelidade.
Ao longo do tutorial, vamos passo a passo — da criação de perfis no Adobe Experience Platform à configuração do canal de email e, finalmente, criar e executar a campanha que seleciona automaticamente a oferta mais relevante usando a tomada de decisão.

## Pré-requisitos para este tutorial

* Acesso ao Experience Platform

* Noções básicas sobre conceitos do Experience Platform (Perfis, Públicos-alvo, Conjuntos de dados)

* Familiaridade com o Journey Optimizer
