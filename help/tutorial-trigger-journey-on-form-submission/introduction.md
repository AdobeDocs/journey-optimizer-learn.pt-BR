# Acionar jornada no envio do formulário

Este caso de uso demonstra como o envio de um formulário de cliente aciona uma jornada do Adobe Journey Optimizer em tempo real, permitindo comunicações de acompanhamento imediatas e personalizadas com base nas informações fornecidas pelo usuário.


## Pré-requisitos para este tutorial

* Acesso ao Experience Platform

* Noções básicas sobre conceitos do Experience Platform (Perfis, Públicos-alvo, Conjuntos de dados)

* Familiaridade com o Journey Optimizer

## Etapas de implementação

* Definir um esquema de evento no Adobe Experience Platform
* Configurar um evento personalizado no Adobe Journey Optimizer
* Configurar canal de email
* Criar uma Jornada acionada pelo evento personalizado
* Enviar dados de envio de formulário para o Adobe Experience Platform
